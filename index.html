Zesjescultuur

Berekenen wat je moet halen voor een proefwerk om een voldoende te staan voor een vak is irritant; je moet alle cijfers in je rekenmachine of een app invoeren om daarna een hele hoop berekeningen uit te voeren, als je het verkeerd doet moet je weer helemaal opnieuw beginnen.

BetterMagister maakt het makkelijk; het laadt alle cijfers voor dat vak automatisch in, en dan hoef je alleen nog maar de weging van de toets en je gewenste gemiddelde in te voeren. Ook kun je berekenen wat je gemiddelde wordt als je een bepaald cijfer haalt, bijvoorbeeld als je het cijfer al terug hebt maar de leraar het nog niet op Magister heeft gezet.

Handleiding

  1. Als je de cijferpagina laadt klik je op een van de berekende gemiddeldes (Onder het kopje 'G-Rx', met x de periode)

  2. In de rechter sidebar verschijnt een tabel met alle cijfers dat je voor dat vak gehaald hebt, onderaan de tabel staat een link 'Mutaties berekenen'. Als deze link niet verschijnt, klik dan op een ander cijfer en weer terug op het gewilde cijfer. Dit is een bug waaraan gewerkt wordt.

  3. Vul de gegevens in en klik op 'Bereken'.

Installatie

BetterMagister is een userscript. Userscripts draaien op een browser addon die userscripts draait. Om BetterMagister te gebruiken moet je dus eerst een van de volgende userscript clients installeren:

  • Mozilla Firefox - Greasemonkey
  • Google Chrome - Tampermonkey
  • Safari - GreaseKit
  • Opera - Tampermonkey of Violent monkey
Disclaimer: BetterMagister is vooral gemaakt voor Greasemonkey, de andere plugins in de lijst zijn ports van Greasemonkey naar andere browsers. Sommige features zouden niet optimaal kunnen werken.

Als je een userscript addon voor jouw browser geinstalleerd hebt, kun je BetterMagister installeren:

  • BetterMagister v1.13 - 30KB
  • BetterMagister v1.13 (Minified) - 24KB

Wanneer je nu inlogt op Magister 6 verschijnt rechtsboven naast je naam een rood instellingen knopje. Als je hierop klikt kun je dingen zoals layout (stylesheet) en agendaweergave (automatische instelling van agendaweergave)


In de toekomst komt er misschien een standalone plugin die je kunt installeren op een browser.

De volledige code en documentatie is te vinden op de BetterMagister GitHub repository.
